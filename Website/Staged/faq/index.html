How can I run a local webserver on my Mac?

Open a terminal window and navigate to the directory where you store your website files. Enter this code to start a server with Phython.

python -m SimpleHTTPServer 8000

Click return and keep the terminal window open.

To look at the website from your Mac type this into a browser window:

http://localhost:8000/
